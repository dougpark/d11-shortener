# Global "Omni-Search"

Instead of just searching titles, make the search bar in your header work across everything.

The Feature: Search titles, summaries, tags, and the full text of your R2 snapshots.

The Perk: Using SQLite FTS5 in D1, this search will be nearly instant, even with 10,000+ entries.

search by bookmark date: 
ex. since april 9, 2023
ex. before april 9, 2023
ex. between april 9, 2023 and may 1, 2023
